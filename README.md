# UmbracoHomework

- Restore database(Check readme file)
- Run project "UmbracoHomework" 
- The project contains two pages: All Submission (List of all submissions) and Submission(Submission Form)
- When creating a submission, you have use a valid e-mail a Date of birth where the user is at least 18 years, 
and then you have to add a serial number which can be found in the database (There have been generated 100 unique serialnumbers).
Valid serialnumbers: 
0AAFE876-CBAB-4D26-8486-EB1577F058B1
ADEA5899-8B0B-44B0-B2FB-EBDAA413493A
04E4FA37-8A7B-4F8D-93A2-EC9B3D8E3210
- When submitting with the same serial number more than twice the serial number is not valid. The serial number is only valid twice.
- If serial number is valid you will be redicred to a new page saying "Congratulations"
